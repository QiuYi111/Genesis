üìÑ Research Abstract

Project Genesis introduces a novel computational framework for studying emergent social complexity through AI-driven agent interactions. We present the Trinity system, a revolutionary approach that enables Large Language Models (LLMs) to dynamically create skills, rules, and social structures based solely on observed agent behaviors, eliminating the need for predefined behavioral hierarchies.

Key Research Questions

  • How can complex social structures emerge from simple agent interactions without predetermined rules?
  • What role do LLMs play in generating contextually appropriate emergent behaviors?
  • Can dynamic skill creation systems produce novel social phenomena?
  • How do cultural norms and governance systems evolve in artificial societies?

Methodology

Our framework employs a 64√ó64 cellular world with heterogeneous terrain, where agents equipped with LLM-based decision-making capabilities interact through natural language. The Trinity system acts as an omniscient observer that identifies behavioral patterns and generates new skills, rules, and environmental changes in response to emergent collective behavior. We conducted extensive simulations across various historical contexts, from Stone Age tribes to Bronze Age civilizations, demonstrating the spontaneous emergence of leadership structures, economic systems, and cultural transmission mechanisms.

Key Findings

  • Social complexity emerges naturally from simple interaction rules without predetermined structures
  • LLM-driven agents develop contextually appropriate behaviors that reflect real-world social dynamics
  • Dynamic skill creation enables novel solutions to environmental and social challenges
  • Cultural transmission mechanisms evolve spontaneously across agent generations
  • Governance systems emerge through consensus-building and resource management behaviors

Research Contribution

This work contributes to computational sociology by demonstrating that complex social phenomena can emerge from simple agent interactions when augmented with LLM-driven contextual understanding. The Trinity system provides a scalable framework for studying social evolution, cultural dynamics, and emergent governance without requiring predefined behavioral architectures.

üöÄ Quick Start Guide

1

Installation

# Clone the repository
git clone https://github.com/QiuYi111/Genesis.git
cd Genesis

# Install dependencies
uv sync

# Install uv if needed
curl -LsSf https://astral.sh/uv/install.sh | sh
2

Configuration

# Set your API key
export DEEPSEEK_API_KEY="your-api-key-here"

# Or for OpenAI
export OPENAI_API_KEY="your-api-key-here"

# Configure simulation parameters
cp sociology_simulation/conf/config.yaml config.yaml
3

Run Simulations

# Stone Age simulation
uv run python -m sociology_simulation.main \
    simulation.era_prompt="Stone Age tribes" \
    world.num_agents=10 \
    runtime.turns=50

# Bronze Age civilization
uv run python -m sociology_simulation.main \
    simulation.era_prompt="Bronze Age civilization" \
    world.num_agents=20 \
    runtime.turns=100
